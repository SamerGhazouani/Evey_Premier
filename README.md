# Evey (Online Events Mobile App)

This project is a mobile application designed to manage and participate in online events efficiently. It provides a complete platform for event attendees, organizers, and sponsors to interact, stay updated, and engage in real-time activities.

# Features

- User Authentication: Secure login and registration for attendees, organizers, and sponsors

- Event Schedule & Planning: View detailed schedules of all activities, including descriptions and speakers

- Participants: Access participant lists and profiles

- Sponsors: View and interact with event sponsors

- Live Stream & Real-Time Chat: Watch live events and chat with other participants instantly

- Quizzes: Participate in interactive quizzes during events
